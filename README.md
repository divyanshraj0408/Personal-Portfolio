## Getting Started

To view the website locally on your machine, follow these steps:

1. Clone this repository: `git clone https://github.com/divyanshraj0408/Personal-Portfolio.git`
2. Navigate to the project directory: `cd personal-portfolio`
3. Run `npm install`.(You should have npm and node already installed in your device)
4. Run `npm start`.

Alternatively, you can check out the live version of my portfolio website at [www.divyanshraj.onrender.com](https://divyanshraj.onrender.com).
